# WriteSage
Welcome to my repository! README before cloning this project! This repository contains the source code of WriteSage, which is a simple blogging site that has been deployed and is accessible online. You can find the link to the deployed site below.\
[Click me to watch it live!](https://writesage.onrender.com/)

## Features
- **User Authentication**: Authors can create accounts, log in, and manage their blog posts.
- **Create and Edit Posts**: Authors can create new blog posts and edit, delete existing ones.
- **Responsive Design**: The site is optimized for mobile devices, ensuring a seamless experience on all screen sizes.

## Local Development

If you want to run this project locally for development or testing purposes, follow these steps:
- Cloning the Repository:
```
git clone git@github.com:satwik-060/WriteSage.git
```
- Install Requirements:
```
pip install -r requirements.txt
```
- Run `run.py`:
```
python run.py
```
- The Website is now deployed locally on `http://127.0.0.1:5000/`
